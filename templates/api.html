API usage instructions

URL:

{{data}}api

KEYS:

required keys for monolingual that should be form-posted:

lng: language code [string]
AND
with_ner: on, if text should be processed using NER ["on"]?
AND
with_nel: on, if text should be processed using NEL ["on"]?
AND
file: file that should be processed [file]
OR
data: string that should be processed [string]

required keys for bilingual processing that should be form-posted:

with_tmx: must be on ["on"]?
AND
with_ner: on, if text should be processed using NER ["on"]?
AND
with_nel: on, if text should be processed using NEL ["on"]?
AND
file: file that should be processed [file]
